# Bael Tree Plantation Donation Platform

## Project Overview
A platform to support Bael tree plantation through donations.

## Tech Stack
Frontend: React + Vite + Tailwind  
Backend: Node.js / Express  
Database: (Supabase / MongoDB if used)

## Project Structure

BaelTrees
│
├── frontend
│   └── React application
│
├── backend
│   └── API server
│
└── README.md

## Setup Instructions

### Run Frontend
cd frontend
npm install
npm run dev

### Run Backend
cd backend
npm install
npm run dev
